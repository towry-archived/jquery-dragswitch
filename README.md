# jquery-dragswitch

jQuery plugin to drag switch elements

# Usage

```javascript
$('.ul').dragswitch(function (drag) {
	drag('.li .box').dragEnd(function () {
		// when drag end	
	}).dragStart(function () {
		// when drag start
	});
});
```
